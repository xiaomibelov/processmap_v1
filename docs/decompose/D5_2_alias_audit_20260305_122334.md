# D5.2 Alias Audit (Backend)

Date: `2026-03-05 12:23:34`
Branch: `feat/decompose-d5-backend-routers-v1`

## Scope

Alias scan source:
- backend routes: `backend/app/_legacy_main.py`
- frontend usage map: `frontend/src/lib/apiRoutes.js`, `frontend/src/lib/api.js`

Pre-change `rg` facts used:
- `frontend/src/lib/apiRoutes.js` had `pathReportsLegacy` / `pathReportLegacy`.
- `frontend/src/lib/api.js` had fallback calls to legacy `.../path/...`.

After D5.2 changes in this branch:
- `pathReportsLegacy` / `pathReportLegacy` were removed from frontend route map.
- reports calls in `frontend/src/lib/api.js` now use canonical `/paths/...` endpoints only.
