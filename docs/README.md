# Baseline Docs Subset

This directory contains only the stable documents that were selectively restored into the baseline candidate.

Included categories:
- user-facing operating guide
- deploy / ops runbooks
- stable API or runtime contracts
- draw.io product and regression-gate specs
- Redis runtime policy and keyspace notes

Intentionally excluded from source `docs/`:
- debug evidence
- forensics and handoff packs
- migration snapshots tied to one cleanup pass
- local audit artifacts
- time-bound decomposition notes without clear long-term repo value

Use `MANIFEST_WORKING_PRODUCT.md` for the full inclusion/exclusion rationale.
