# 16_Журнал решений

## 2026-05-07 — preview UI before apply

Decision: expose `ai.process.extract_from_notes` as a contextual preview UI in the notes surface, but do not provide an enabled apply control until the backend apply-boundary contour exists.

Rationale:

- backend preview endpoint is available and non-mutating;
- legacy `POST /notes` still mutates session truth and must not be called by preview action;
- user must see fallback/source warnings and candidate diff before domain truth changes;
- apply behavior needs a separate CAS-backed endpoint and contour.

Explicit unchanged: backend behavior, DB/schema, prompts, product actions, BPMN XML, export, merge/deploy/PR.
