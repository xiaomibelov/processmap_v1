| Поле | Значение |

Frontend now exposes a preview-only review panel for notes extraction. The panel calls the dedicated preview endpoint and renders `source`, `warnings`, `candidate_roles`, `candidate_start_role`, `candidate_nodes`, `candidate_edges`, `candidate_questions`, `diff`, and `input_hash`.

Hard boundary:

- preview action does not call legacy `POST /api/sessions/{sid}/notes`;
- preview action does not mutate local session truth;
- apply button is disabled with copy that apply is a later contour;
- legacy notes endpoint behavior remains unchanged.
