# 16_Журнал решений

## 2026-05-07 — explicit apply for notes extraction

Decision: `ai.process.extract_from_notes` apply is a separate endpoint from preview and legacy notes save. Apply accepts selected candidates and strict CAS; it does not invoke AI, DeepSeek or fallback parsing.

Rationale:

- preview output is suggestion state, not domain truth;
- accepted candidates must be explicit;
- CAS prevents stale graph overwrite;
- legacy `/notes` remains temporarily unchanged for compatibility.

Explicit unchanged: prompts, preview AI behavior, product actions, BPMN XML, export, merge/deploy/PR.
