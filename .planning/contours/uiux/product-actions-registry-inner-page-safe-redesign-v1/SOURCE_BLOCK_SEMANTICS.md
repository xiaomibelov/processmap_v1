# SOURCE_BLOCK_SEMANTICS — семантика блока источника/сессии

**Контур:** `uiux/product-actions-registry-inner-page-safe-redesign-v1`  
**Run ID:** `20260517T134517Z-85981`  
**Agent:** Agent 3 / Executor Part 2  
**Дата:** `2026-05-17`

## Цель

Зафиксировать, что project/session controls являются выбором источника данных для реестра, а не действиями над строками таблицы и не новым navigation shell.

## Проверяемые утверждения

| # | Утверждение | Ожидаемый результат | Проверка |
|---|---|---|---|
| S.1 | «Открыть проект» открывает проект как источник данных для реестра | Пользователь понимает, что меняется source scope registry dataset | Click path + визуальный контекст |
| S.2 | «Открыть сессию» открывает сессию как источник данных | Пользователь понимает, что меняется source scope registry dataset | Click path + визуальный контекст |
| S.3 | Кнопки источника не путаются с row actions | Controls находятся в отделенной section, не внутри registry rows | Screenshot/DOM |
| S.4 | Sources section имеет ясный title | «Источники данных» или «Выбор источника» | Визуально |
| S.5 | Sources section secondary | Блок не конкурирует с title/metrics/table и не занимает верхнюю треть экрана | Fresh 1920x1080 screenshot |

## Рекомендуемые подписи

| Текущая подпись | Предпочтительная подпись | Почему |
|---|---|---|
| «Открыть проект» | «Загрузить из проекта» | Говорит о загрузке registry dataset, а не generic navigation |
| «Открыть сессию» | «Загрузить из сессии» | Уточняет source semantics |

Допустимые альтернативы, если они лучше совпадают с существующим продуктовым языком:

- «Выбрать проект» / «Выбрать сессию»
- «Источник: проект» / «Источник: сессия»
- «Использовать проект» / «Использовать сессию»

## Рекомендуемый section title

- Основной вариант: «Источники данных».
- Альтернатива: «Выбор источника».
- Не рекомендуется: title отсутствует или выглядит как обычная строка таблицы.

## Fail conditions

- Source controls расположены до таблицы как primary area и перетягивают внимание.
- Source rows визуально неотличимы от registry rows.
- Labels «Открыть проект» / «Открыть сессию» стоят без контекста source section.
- Click по source action выглядит как действие над выбранной строкой таблицы.
