# REVIEW_PASS superseded

Актуальный статус contour: `CHANGES_REQUESTED`.

Любые старые `REVIEW_PASS`, `READY_FOR_REVIEW`, runtime screenshots или review reports по этому contour считаются историческими evidence и не являются текущим release/merge gate.

Причина supersede:

- empty workspace scope может выглядеть сломанным;
- AI controls размещены в secondary sources area вместо primary filters/actions area;
- dirty workspace и non-canonical checkout должны быть классифицированы до final pass.
