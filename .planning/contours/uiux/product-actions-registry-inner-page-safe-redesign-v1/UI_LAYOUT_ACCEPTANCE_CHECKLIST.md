## 3. Ряд метрик

- [ ] Метрики расположены **прямо под заголовком/табами**, а не внизу страницы.
- [ ] Видны 5 метрик: Сессии, Строк, Полных, Неполных, После фильтров.
- [ ] Значения метрик — реальные (или «—» при отсутствии данных), не placeholder.

- [ ] Фильтры расположены **горизонтально** или в компактной **сетке**.
- [ ] Не используется вертикальный стек фильтров по левому краю.

- [ ] Кнопка «Сбросить фильтры» видна и доступна.
- [ ] Счётчик «Выбрано для AI: N / 10» виден.

- [ ] Кнопки экспорта CSV/XLSX видны (если не перенесены в заголовок).
- [ ] Панель фильтров не толкает таблицу за пределы первого экрана.

## 5. Предупреждающий баннер

- [ ] Баннер показывается **только при наличии** неполных строк.
- [ ] Баннер расположен **над таблицей**.
- [ ] Текст баннера корректен и читаем.
- [ ] Баннер не дублируется.

## 6. Главная таблица

- [ ] Таблица является **доминирующим элементом** страницы.
- [ ] 4 колонки: Продукт, Действие, Процесс / шаг, Статус.
- [ ] Строки компактные и читаемые.
- [ ] Бейджи статуса: «Полная» / «Неполная».
- [ ] Чипы/теги выровнены, не агрессивны по цвету.
- [ ] Пагинация видна внизу таблицы.
- [ ] Размер страницы переключается (25 / 50), макс. 50.
- [ ] Горизонтальный скролл таблицы минимален или отсутствует при нормальной ширине.

- [ ] Подписи «Открыть проект» / «Открыть сессию» однозначно говорят об источнике, а не о строке реестра.
